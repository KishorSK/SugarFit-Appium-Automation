# Sugar.fit Mobile Automation Assignment

## Technology Used

- Java
- Appium
- TestNG
- Maven
- Page Object Model (POM)
- Extent Reports

## Project Features

- App Launch Validation
- Login Validation
- Meal Logging Validation
- Screenshot on Failure
- Extent HTML Report

## How to Run

1. Start Appium Server
2. Connect Android device
3. Run testng.xml
